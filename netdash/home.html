Network Performance Dashboard

Description

The Network Performance Dashboard is a web application that provides an interactive and user-friendly interface for visualizing and analyzing network performance metrics for different Huawei clients. The dashboard will allow users to select specific clients, view performance trends over time, and compare rankings across different months.

Key Features

Customer Selection

Time Range Selection

Ranking Comparison

Interactive Visualization
